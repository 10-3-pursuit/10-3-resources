1. ## Native Array Methods
    1. `.map` - loop through a complete array and return a new array with values.
        - [.map part 1](https://drive.google.com/file/d/1ZIcRCoF__0C4cEWfJmZ3qHcJwnReJEUO/view?usp=drive_link)
